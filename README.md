# Half-Wave-Rectifier-LTSpice

### About
Simulation of half wave rectifier 
using D1N4007 diode with LED indicator.

### Tools Used
- LTSpice

### Components
- D1N4007 Diode
- LED (D2)
- Resistor 10k
- AC Source SINE(0 325 50)
